NFT Haze
art information
Description
This is the first piece in a series that expands Kristen's Cuboid Vortex series into cityscapes. Created with vintage paint software from 1990-91: Ultrapaint for 68k Macintosh and Deluxe Paint IV for Commodore Amiga. This work uses colour range/palette cycling, and speaks to the history of paint software for early personal computers. Kristen's work is a living document of the processes associated with historic software for early personal computers. Created in the standard Amiga scale 320x256(x4), 24fps, 2021
